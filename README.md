# Discord Optimization Script 🚀

這是一個專為 Windows 使用者設計的輕量級批次檔腳本，能夠一鍵自動為您的 Discord 安裝 **OpenAsar** 與 **Vencord**。

本腳本經過優化，具備「智慧快取機制」，在大多數情況下能實現零延遲啟動，且不佔用額外網路資源。

## ✨ 核心功能

* **極速啟動**：自動檢查本地檔案日期。若檔案更新未滿 7 天，將跳過 GitHub 連線，瞬間完成注入。
* **自動化環境準備**：自動管理 `C:\DiscordMods` 資料夾作為本地快取與工具存儲點。
* **安裝 OpenAsar**：優化 Discord 核心，大幅提升啟動速度與反應效能。
* **安裝 Vencord**：自動注入 Vencord 插件系統，解鎖自訂主題與功能增強。
* **智慧維護**：自動處理 Discord 官方更新後的覆蓋問題，確保優化始終有效。

## ⚠️ 事前準備 (重要!)

在使用本腳本之前，**請務必關閉 Discord 內建的開機啟動設定**，以避免與腳本產生衝突：

1. 開啟 Discord，點擊左下角的「使用者設定」(齒輪圖示)。
2. 在左側選單中找到「Windows 設定」。
3. 將 **「開機時啟動 Discord」** 選項**關閉**。

*註：未來將由我們的腳本負責引導 Discord 啟動，這樣才能確保每次啟動都包含優化補丁。*

## 🛠️ 如何使用

1. 下載本專案中的 `Discord-Optimization.bat`。
2. 雙擊執行 `Discord-Optimization.bat` 即可完成首次安裝與優化。

## 🔄 設定開機自動優化 (推薦)

由於 Discord 官方更新時常會覆蓋掉優化檔案，將腳本加入開機啟動可以確保您的 Discord 永遠處於最佳狀態。

1. 按下 `Win + R`，輸入 `shell:startup` 並 Enter。
2. 將 `Discord-Optimization.bat` 的**捷徑**放入該資料夾中。
3. 之後每次開機，腳本會自動閃電檢查版本、套用優化並啟動 Discord。

## ⚙️ 腳本行為說明

* **首次執行**：會連線至 GitHub 下載必要的組件並存儲於 `C:\DiscordMods`。
* **日常啟動**：腳本會檢查本地檔案，若距離上次下載未滿 7 天，則直接使用本地檔案，不產生任何網路延遲。
* **強制更新**：若想立刻抓取最新版，只需手動刪除 `C:\DiscordMods` 資料夾後再次執行即可。
